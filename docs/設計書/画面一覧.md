# 画面一覧

> 更新日: 2026-04-01

## タブ画面（4タブ + 1非表示）

| タブ | ルート | 画面名 | 概要 |
|------|--------|--------|------|
| 1 | `/(tabs)/index` | ホーム | ちかく+フィード統合。距離帯セクション / カテゴリ・日付フィルタ。モード切替 |
| 2 | `/(tabs)/talk` | タイムライン | トーク一覧。24hゴースト + 殿堂入り。LIVE/殿堂タブ |
| 3 | `/(tabs)/ai` | さがす | テキスト検索 + 急上昇 + 週間ランキング + おすすめフィード |
| - | `/(tabs)/feed` | フィード | 日付ピッカー付きフィード。ホームからも遷移可能（タブ非表示） |
| 4 | `/(tabs)/settings` | 設定 | プロフィール + テーマ切替 + 文字サイズ + ログアウト |

## モーダル画面

| ルート | 画面名 | 概要 |
|--------|--------|------|
| `/post` | 投稿作成 | カテゴリ → タイトル → 詳細 → 期限 → 写真 → 場所 |
| `/talk-post` | トーク投稿 | 140文字 + 写真 + 位置情報 |

## スタック画面

| ルート | 画面名 | 概要 |
|--------|--------|------|
| `/feed/[id]` | 投稿詳細 | ヒーロー画像 + コメント + いいね + 地図（遅延読込） |
| `/feed-edit` | 投稿編集 | 既存投稿の編集 |
| `/talk-detail/[id]` | トーク詳細 | スレッド表示 + 返信入力 |
| `/profile/[id]` | プロフィール | アバター + 統計 + バッジ + 投稿/トークタブ + フォロー |
| `/profile/edit` | プロフィール編集 | アバター・名前・自己紹介・場所・プライバシー設定 |
| `/follow-list/[id]` | フォロー一覧 | フォロワー/フォロー中のユーザーリスト |
| `/street-history` | 街の記憶 | 位置ベースの過去投稿を月別グループ表示 |
| `/notifications` | 通知一覧 | いいね・コメント・返信・フォローの通知リスト |

## 認証画面

| ルート | 画面名 |
|--------|--------|
| `/(auth)/login` | ログイン |
| `/(auth)/signup` | 新規登録 |

## グローバルUI（全画面共通）

| コンポーネント | 配置 | 概要 |
|---------------|------|------|
| `Header` | タブ上部 | 「n」ロゴ + navios + 通知ベル（未読バッジ）+ アバター |
| `Fab` | タブ右下 | 投稿 / トーク作成メニュー。ゲストガード付き |
| `OnboardingTour` | ルート | 初回起動時3ステップガイド |
| `GuestLoginSheet` | ルート | ゲスト操作時のログイン誘導ボトムシート |
| `Toast` | ルート | 右からスライドインの通知トースト |
| `OfflineBanner` | ルート | ネットワーク切断時のバナー |
