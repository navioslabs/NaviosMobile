# 画面一覧

> 更新日: 2026-03-28

## タブ画面（5タブ）

| タブ | ルート | 画面名 | 概要 |
|------|--------|--------|------|
| 1 | `/(tabs)/index` | ホーム | ちかく+フィード統合。距離帯セクション / カテゴリ・日付フィルタ。モード切替 |
| 2 | `/(tabs)/talk` | この街の声 | 吹き出し型トーク一覧。24hゴースト + 殿堂入り。LIVEバッジ |
| 3 | `/(tabs)/ai` | さがす | テキスト検索 + 急上昇 + 週間ランキング + おすすめフィード |
| 4 | `/(tabs)/feed` | ※未使用 | ホーム画面に統合済み |
| 5 | `/(tabs)/settings` | マイページ | プロフィール（アバター・活動サマリー）+ 表示設定 + ログアウト |

## モーダル画面

| ルート | 画面名 | 概要 |
|--------|--------|------|
| `/post` | 投稿作成 | カテゴリ選択 → タイトル → 詳細 → 期限 → 写真 → 場所（Lv1 GPS / Lv2 Places検索） |
| `/talk-post` | トーク投稿 | 140文字 + 写真 + 位置情報。3段階送信ボタン |

## スタック画面

| ルート | 画面名 | 概要 |
|--------|--------|------|
| `/feed/[id]` | 投稿詳細 | ヒーロー画像（カテゴリ色グラデ）+ コメント一覧 + 入力バー + 地図（Lv3遅延） |
| `/talk-detail/[id]` | トーク詳細 | スレッド表示 + 返信入力 + プロフィールポップオーバー |
| `/profile/[id]` | プロフィール | アバター + 統計 + バッジ + 投稿/トークタブ切替 |
| `/profile/edit` | プロフィール編集 | アバター・名前・自己紹介・場所・プライバシー設定 |
| `/street-history` | 街の記憶 | 位置ベースの過去投稿を月別グループ表示 |
| `/notifications` | 通知一覧 | いいね・コメント・返信の通知リスト |

## 認証画面

| ルート | 画面名 |
|--------|--------|
| `/(auth)/login` | ログイン |
| `/(auth)/signup` | 新規登録 |

## グローバルUI（全画面共通）

| コンポーネント | 配置 | 概要 |
|---------------|------|------|
| `Header` | タブ上部 | 「n」ロゴ + navios + 通知ベル（未読バッジ）+ アバター |
| `Fab` | タブ右下 | 投稿 / トーク作成メニュー。ゲストガード付き |
| `OnboardingTour` | ルート | 初回起動時3ステップガイド |
| `GuestLoginSheet` | ルート | ゲスト操作時のログイン誘導ボトムシート |
| `Toast` | ルート | 右からスライドインの通知トースト |
| `OfflineBanner` | ルート | ネットワーク切断時のバナー |
