# データベース設計

> 更新日: 2026-04-01

## テーブル一覧

| テーブル | 用途 | RLS |
|---------|------|-----|
| `profiles` | ユーザープロフィール（Auth連動） | SELECT: 公開 or 自分 / UPDATE: 自分 |
| `posts` | フィード投稿（ライフライン/イベント/近助） | SELECT: 全員 / INSERT,UPDATE,DELETE: 自分 |
| `talks` | トーク（140文字、24hゴースト） | SELECT: 全員 / INSERT,UPDATE,DELETE: 自分 |
| `comments` | 投稿へのコメント | SELECT: 全員 / INSERT,DELETE: 自分 |
| `talk_replies` | トークへの返信 | SELECT: 全員 / INSERT,DELETE: 自分 |
| `likes` | いいね（post/talk/comment/reply） | SELECT: 全員 / INSERT,DELETE: 自分（UNIQUE制約） |
| `reports` | 通報 | SELECT: 自分 / INSERT: 自分（UNIQUE制約） |
| `user_badges` | ローカルレジェンドバッジ（resident/face/legend） | SELECT: 全員 |
| `notifications` | アプリ内通知（like/comment/reply/hall_of_fame/follow） | SELECT,UPDATE: 自分 |
| `follows` | フォロー関係 | SELECT: 全員 / INSERT,DELETE: 自分（UNIQUE制約） |
| `maps_api_logs` | Google Maps API呼び出しログ | INSERT: 認証ユーザー |

## RPC関数

| 関数 | 用途 |
|------|------|
| `get_nearby_posts(user_lat, user_lng, radius_m)` | PostGIS距離計算で近隣投稿を取得 |
| `get_post_detail(post_id, user_lat, user_lng)` | 投稿詳細（距離計算付き） |
| `get_feed_posts(user_lat, user_lng, category_filter, created_after, created_before, page_offset, page_limit)` | フィード一覧（ページネーション + 距離計算） |
| `get_street_history(user_lat, user_lng, radius_m)` | 位置ベースの過去投稿履歴（殿堂Talk + 人気Post） |
| `get_daily_digest(p_user_id, p_user_lat, p_user_lng)` | デイリーダイジェスト（トレンド・ハイライト・統計を1回で取得） |
| `get_trending_tags(user_lat, user_lng, limit)` | トレンドタグ取得 |
| `refresh_user_badges(target_user_id)` | バッジ再計算 |
| `get_user_top_badge(target_user_id)` | ユーザーの最上位バッジ取得 |
| `search_posts_trgm(search_query, category_filter, result_limit)` | pg_trgm トリグラム類似度検索 |
| `cleanup_expired_posts()` | 期限切れ投稿の自動削除 |

## トリガー

| トリガー | テーブル | 動作 |
|---------|---------|------|
| `on_auth_user_created` | auth.users | 新規ユーザー → profiles 自動作成 |
| `on_like_insert` | likes | いいね → notifications に通知作成（自分除外） |
| `on_comment_insert` | comments | コメント → 投稿者 + 他コメント者に通知作成 |
| `on_reply_insert` | talk_replies | 返信 → Talk投稿者に通知作成 |
| `on_follow_insert` | follows | フォロー → フォロー対象に通知作成 |
| `promote_to_hall_of_fame` | likes | 10いいね到達 → talks.is_hall_of_fame = true |
