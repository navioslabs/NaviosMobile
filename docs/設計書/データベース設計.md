# データベース設計

> 更新日: 2026-03-28

## テーブル一覧

| テーブル | 用途 | RLS |
|---------|------|-----|
| `profiles` | ユーザープロフィール（Auth連動） | SELECT: 公開 or 自分 / UPDATE: 自分のみ |
| `posts` | フィード投稿 | SELECT: 全員 / INSERT,UPDATE,DELETE: 自分のみ |
| `talks` | トーク（140文字短文） | SELECT: 全員 / INSERT,UPDATE,DELETE: 自分のみ |
| `comments` | 投稿へのコメント | SELECT: 全員 / INSERT,UPDATE,DELETE: 自分のみ |
| `talk_replies` | トークへの返信 | SELECT: 全員 / INSERT,UPDATE,DELETE: 自分のみ |
| `likes` | いいね（投稿/トーク/コメント/返信） | SELECT: 全員 / INSERT,DELETE: 自分のみ |
| `reports` | 通報 | SELECT,INSERT: 自分のみ |
| `user_badges` | ローカルレジェンドバッジ | SELECT: 全員 |
| `notifications` | アプリ内通知 | SELECT,UPDATE: 自分のみ |

## RPC関数

| 関数 | 用途 |
|------|------|
| `get_nearby_posts(user_lat, user_lng, radius_m)` | PostGIS距離計算で近隣投稿を取得 |
| `get_street_history(user_lat, user_lng, radius_m)` | 位置ベースの過去投稿履歴 |
| `refresh_user_badges(target_user_id)` | バッジ再計算 |
| `get_user_top_badge(target_user_id)` | ユーザーの最上位バッジ取得 |
| `search_posts_trgm(search_query, category_filter, result_limit)` | pg_trgm トリグラム類似度検索 |

## トリガー

| トリガー | テーブル | 動作 |
|---------|---------|------|
| `on_auth_user_created` | auth.users | 新規ユーザー → profiles 自動作成 |
| `on_like_insert` | likes | いいね → notifications に通知作成 |
| `on_comment_insert` | comments | コメント → notifications に通知作成 |
| `promote_to_hall_of_fame` | likes | 10いいね到達 → talks.is_hall_of_fame = true |

## Storage バケット

| バケット | 用途 | アクセス |
|---------|------|---------|
| `avatars` | プロフィール画像 | 全員閲覧 / 自分のみUP・削除 |
| `post-images` | 投稿画像（最大3枚） | 全員閲覧 / 認証ユーザーUP |
| `talk-images` | トーク画像 | 全員閲覧 / 認証ユーザーUP |
