# Androidエミュレータ強制終了手順

## 症状

- Android Emulatorが通常の方法（×ボタン、AVD Manager）で終了できない
- PCを再起動しても自動的にエミュレータが起動してしまう

## 対処法

### 1. 関連プロセスの確認

```bash
tasklist | findstr /i "qemu emulator adb"
```

以下の3プロセスが対象:

| プロセス名               | 役割                                       |
| ------------------------ | ------------------------------------------ |
| `qemu-system-x86_64.exe` | エミュレータ本体（最もメモリ消費が大きい） |
| `emulator.exe`           | エミュレータランチャー                     |
| `adb.exe`                | Android Debug Bridge                       |

### 2. 全プロセスを強制終了

```bash
taskkill /F /IM qemu-system-x86_64.exe
taskkill /F /IM emulator.exe
taskkill /F /IM adb.exe
```

`/F` は強制終了オプション。

### 3. 終了確認

```bash
tasklist | findstr /i "qemu emulator adb"
```

何も表示されなければ完了。

## 再発防止

再起動後も自動起動する場合は以下を確認:

1. **Windowsスタートアップ**: `タスクマネージャー > スタートアップ` タブでエミュレータ関連を無効化
2. **Android Studio設定**: `File > Settings > Tools > Emulator` で「Launch in a tool window」のチェックを確認
3. **AVD設定**: Android Virtual Device Managerで不要なデバイスを削除
