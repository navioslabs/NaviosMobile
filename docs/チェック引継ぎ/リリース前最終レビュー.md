## 1. プロジェクト概要

━━━━━━━━━━━━━━━━━━━━━━━━━━━━

- **アプリ名**: NaviOs（ナビオス）— 「近くを知る。暮らしが変わる。」
- **種別**: クロスプラットフォームモバイルアプリ（iOS / Android / Web）
- **技術スタック**: Expo ~54 + React Native 0.81 + TypeScript 5.9 + Supabase (Auth, Postgres, PostGIS, Storage) + React Query + Zustand + React Native Reanimated
- **管理画面**: NaviosAdmin — Hono + React 19 SPA + Cloudflare Pages（別リポジトリ: navioslabs/NaviosAdmin）
- **対象ユーザー**: 地域住民（一般消費者向け）— 日本語のみ対応
- **主要機能**:
  - **ちかく（Nearby）**: GPS連動で距離別に周辺投稿を表示（200m / 500m / それ以上）
  - **タイムライン（Talk）**: 140文字のリアルタイムつぶやき（24時間で消える「ゴースト投稿」＋10いいねで殿堂入り）
  - **さがす（Search/Discover）**: トリグラム検索 + トレンド + 週間ランキング + AIサジェスチョン
  - **フィード（Feed）**: カテゴリ別・日付別の投稿一覧（lifeline / event / help）
  - **バッジシステム**: 投稿・いいね数に基づくエリアバッジ（住民 / 顔役 / レジェンド）

## 6. 良い点

━━━━━━━━━━━━━━━━━━━━━━━━━━━━

4. **Skeleton完成度** — Reanimatedパルスアニメーション付き

6. **アニメーション作り込み** — Reanimated 50+箇所でネイティブ級UX
7. **Zustand使い分け** — UI状態5ストアに限定、サーバーデータ混入なし
