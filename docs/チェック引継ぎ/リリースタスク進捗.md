### 説明文

```
NaviOs（ナビオス）は、あなたの「今・ここ」に関係する地域情報だけを届けるアプリです。

【ちかく】
半径200m・500mの距離で区切って、近くの情報をリアルタイムに表示。歩くだけでリストが変わります。

【タイムライン】
140文字のつぶやきで、今この瞬間の街の空気を共有。24時間で消える「ゴースト投稿」、10いいねで永久保存の「殿堂入り」。

【さがす】
気になるキーワードで地域の投稿を検索。トレンドや週間ランキングもチェックできます。

【フィード】
物資・イベント・近助・行政の4カテゴリで、日付ごとに地域の出来事を一覧。

遠くのニュースは届くのに、隣で何が起きているか知らない。
NaviOsは「検索」を前提にしません。開くだけで、街が見えます。
```
