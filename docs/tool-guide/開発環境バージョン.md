| `nativewind` + `tailwindcss` | classNameがiOS/AndroidのTextで不安定。style prop統一に移行 |

## スタイリング方針

- **NativeWindは使用しない** — classNameではなく、全て `style` prop で統一
- テーマカラーは `makeTokens(isDark)` 経由で動的に取得し `style` に渡す
- テーマ状態は `useThemeStore`（Zustand）で管理

## 補足

- `npx eas-cli --version` は、現在の sandbox 環境では npm cache 制約により直接確認できませんでした。
- そのため `eas-cli` は、インストール確認時に表示された `18.4.0` と、`eas.json` に定義されている最小要件 `>= 18.4.0` を記録しています。
- 現在の EAS 設定ファイルは `eas.json` です。
